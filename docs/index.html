Built to actually replace it

Not a skin. A ground-up native tool that does the whole job and then keeps going where Task Manager stops.

Live process list

Every process with live CPU, memory, threads, PID and responding status. Sort by any column, and end a task with a click or a double-tap.

Performance graphs

A full-history CPU graph with a per-logical-core grid, plus memory, GPU, disk and network — each with its own live, gridded graph.

GPU, power & temps

On NVIDIA cards, read utilization, memory, temperature, power draw in watts, core clock and fan speed straight from the driver — figures Task Manager never shows.

Startup control

See every startup app — from the Run keys and Startup folders — with its publisher, and flip user-scoped ones on or off with a single toggle.

Be your default

One switch in Settings makes Ctrl+Shift+Esc, the taskbar menu and Run → taskmgr all open Pulse instead. Fully reversible — flip it back anytime.

Private & open

No accounts, no telemetry, no network calls. Open source under MIT so you can read exactly what it does. It runs on your machine, for you.

Get started in a minute

Install Pulse

Two ways to run it. The installer is easiest; the portable zip needs no install at all.

Installer Recommended

  1. Download Pulse-Setup.exe from the latest release.
  2. Run it — if SmartScreen warns "unknown publisher," click More info → Run anyway (this build isn't code-signed yet).
  3. It installs per-user — no admin prompt — and adds a Start-menu shortcut.
  4. Optional: in Settings, flip "Make Pulse the default Task Manager" so Ctrl+Shift+Esc opens it.
